然后圣灵开始问我：“你现在觉得怎么样呢？”，我回答说“我非常愤怒你难道感觉不到吗？我都不想见到你，感受到你，你难道不知道吗？”  
在那时间我才意识到，虽然信主这么多年，但是我一直觉得，我只有在*感觉很好的时候*才能见神，但当我软弱害怕的时候我**逃避神**。  
但是圣灵想让我知道，神要我敞开我的*全部的心*来面对他。从此以后我就有了一个**极大的安全感**。

`**没有什么**能够拦阻我**回到神的爱里面**！`

**罪**不能拦阻我、**仇敌**不能拦阻我、甚至**我自己**也不能拦阻我*回到神里面*。
当我*软弱*了又如何，神说，祂要在我软弱的时候，住在我里面。即使*跌倒*了又如何，神住在我里面。

所以即使我还是很软弱，但是我向神祷告说，愿你的光更多进入到我里面来。我不会再逃避你了。  
我要住在你里面，因为**这是你想要的**。
